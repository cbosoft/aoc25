# Day 8
`Java` - corporate easy mode.

Requires Java (e.g. [Open JDK](https://openjdk.org/)).

```bash
cat input.txt | java p1.java
cat input.txt | java p2.java
```

# Part 1
Connect points to networks a set number of times. Multiply top 3 network sizes.

# Part 2
Connect points to networks until there is only one left. Return product of x values of last connection to be made.

# Thoughts on Java
Ah I've missed having a fully featured language for these last few problems. It was easy to solve the problem greedily but it was also easy to fix the issue and speed things up. The hefty standard library was much appreciated.
